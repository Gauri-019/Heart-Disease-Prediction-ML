# Heart-Disease-Prediction-ML
# ❤️ Heart Disease Prediction System

## 📌 Overview
This project predicts the risk of heart disease using Machine Learning algorithms. It takes patient health data as input and provides prediction along with probability and graphical analysis.

## 🚀 Features
- Predicts heart disease risk (High / Low)
- Uses Logistic Regression & Random Forest
- Displays probability of disease
- Graph-based output:
  - Model accuracy comparison
  - Prediction probability
  - Feature importance
- Simple user input interface

## 🧠 Algorithms Used
- Logistic Regression
- Random Forest Classifier
